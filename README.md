# fenlason-hack
The original Hack source code submitted by Jay Fenlason and classmates for inclusion on the USENIX 1982-1 tape.

This source code was created primarily by Jay Fenlason, with additional contributions from Kenny Woodland, Mike Thome, and Jon Payne. It is being shared here under a [CC-BY-NC-SA 4.0 license](https://creativecommons.org/licenses/by-nc-sa/4.0/), as that is the closest modern license to the original distribution license.
